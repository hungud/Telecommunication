Проверка автоподключения интернет-опций

В зависимости от скорости расходования трафика клиента определяются интервалы последующих проверок автоподключения интернет-опций. Если более 2-х часов не осуществлялась проверка на автоподключение интернет-опции, то штатная работа автоподключения по данному номеру нарушилась.

Причинами нарушения штатной работы автоподключения интернет-опций по номеру могут быть:

1. Отсутствие данных по остаткам пакетов;

Данные отсутствуют по причине того, что их не передает билайн. С такими номерами ничего не делаем, они так и остаются висеть в очереди проверки автоподключения пакетов до тех пор, пока по ним не вернуться остатки от билайна.

2. Интернет-опция алгоритмом автоподключения была определена и занесена в историю подключений, но не подключена по некоторым причинам;

Если:

- интернет-опция присутствует в истории подключения интернет-опций;

- присутствует статистика использования интернет-опции;

- заявка на подключение интернет-опции поставлена пользователем CORP_MOBILE;

- отсутствуют остатки по интернет-опции,

то алгоритм автоподключения совершал подключение данной интернет-опции, но по некоторым причинам опция не подключилась.

Подключаем интернет-опцию через карточку абонента (нажать кнопку "Показать информацию по номеру" или двойной щелчок левой кнопкой мыши по номеру).  

3. Номер сменил тарифный план (ТП) с ТП с признаком автоподключения пакетов на ТП с признаком автоподключения пакетов.

Если:

- новый ТП отсутствует в истории подключения интернет-опций;

- по новому ТП отсутствует статистика использования;

- новый ТП присутствует в остатках по пакетам,

то необходимо внести новый ТП в алгоритм автоподключения интернет-опции (нажать кнопку "Добавить интернет-опцию в логику автоподключения").

 

Если:

- по номеру возвращаются остатки по подключенным пакетам;

- в остатках по пакетам имеются остатки по последней подключенной интернет-опции;

- не производилась смена тарифного плана,

то номер ожидает проверки на автоподключение.

В таких случаях можно осуществить проверку на автоподключение интернет-опции вручную, для этого необходимо нажать кнопку "Запустить проверку на автоподключение по номеру". 

Для запуска ручной проверки на автоподключение по всем номерам, присутствующим на вкладке "Очередь на проверку автоподключений интернет-опций", необходимо нажать кнопку "Запустить проверку на автоподключение по всем номерам".
